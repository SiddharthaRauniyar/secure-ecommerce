# Secure E-commerce Website with Vulnerability Testing

## Project Overview
This project is a secure e-commerce web application developed as part of a final-year computing and cybersecurity project. The system simulates a small online shopping platform where users can register, log in, browse products, add items to a cart, place orders, and manage their accounts. The project also focuses on secure development practices and vulnerability testing.

## Features
- User registration and login
- Product catalogue
- Shopping cart
- Checkout and order creation
- Order history
- Order cancellation
- Custom admin dashboard
- Admin order management
- Email notifications
- Dashboard charts
- Security-focused design

## Security Features
- Authentication and authorization
- Admin-only dashboard
- Input validation
- Session protection
- CSRF protection
- Email credentials stored in environment variables

## Technology Stack
- Python
- Django
- SQLite
- HTML
- CSS
- Bootstrap
- Chart.js

## Setup
1. Clone the repository
2. Create a virtual environment
3. Install dependencies
4. Create a `.env` file
5. Run migrations
6. Create a superuser
7. Run the server
